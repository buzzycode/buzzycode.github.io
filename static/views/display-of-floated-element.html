"display" of Floated Element

Today when I was playing with float, I found that I was able to apply margin, padding, width & height to floated element even though the element was an inline element which made me curious to peep under the hood. So I delved more and found that float dictates it's own behvaiour regards to some properties eg. width & height and overrides display property.

For an element, whenever float is set to left or right, float sets its display value to block(if it's not none) but it dictacts it's own rule. eg. if no width is provided then an standard block element takes 100% width of parent but in case of float it takes as much as it needs.
